# שיווק — ניהול משלוחים

דשבורד לניהול משלוחים ומכירות סיטונאיות (ברכסוני + העברות GB).

- **אתר:** GitHub Pages (הקובץ `index.html`)
- **נתונים:** נשמרים בגוגל שיטס פרטי דרך Google Apps Script (`Code.gs`)
- אין נתונים עסקיים בקוד — הכל בגיליון הפרטי.

## הקמה חד-פעמית

1. פותחים [גוגל שיטס חדש](https://sheets.new) בשם "שיווק - נתונים".
2. תפריט **תוספים (Extensions) → Apps Script**.
3. מוחקים את הקוד שיש שם ומדביקים את כל התוכן של `Code.gs`.
4. **פריסה (Deploy) → New deployment → Web app**:
   - Execute as: **Me**
   - Who has access: **Anyone**
5. מאשרים הרשאות, מעתיקים את כתובת ה-**exec**.
6. באתר → ⚙️ הגדרות → מדביקים את הכתובת → שמור.

(את ההגדרה מזינים פעם אחת בכל מכשיר — טלפון/מחשב.)
